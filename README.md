# DeLand Ventures

A modern landing page for DeLand Ventures, a real estate and technology investment firm based in Ann Arbor, MI.

## Features
- Responsive design
- Tech-forward aesthetic
- Portfolio section highlighting 2734 Hampshire Rd
- Clean, semantic HTML5 and Vanilla CSS
